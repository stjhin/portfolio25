## [Hopper](https://stjhin.github.io/hopper/)

This is using [Punk API](https://github.com/alxiw/punk-paging) as back-end. It allows user to type any food and pulls all beer that pairs good with the food.

## [Listy](https://stjhin.github.io/listy)

This is a pure JS project to create your own grocery list with calculator to know how much your total grocery will be.
